# UzLeetCode

UzLeetCode is an AI-powered coding platform that removes language barriers by providing
Uzbek translations of coding problems and instant AI-based code critique.

## Tech Stack
- FastAPI (Python)
- SQLite + SQLAlchemy
- Gemini API (AI Judge & Translation)
- Pure HTML + Tailwind CSS
